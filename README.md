## 🚀 Quick Start Guide

### Prerequisites
- Python 3.8 or higher
- Git installed on your system
- Internet connection for AI API access

### Installation Steps

```bash
# 1. Clone the repository
git clone https://github.com/LokeshChowdary05/FinHealth-Agent.git
cd FinHealth-Agent

# 2. Create virtual environment
python -m venv env

# 3. Activate virtual environment
# On Windows:
.\env\Scripts\Activate.ps1
# On macOS/Linux:
source env/bin/activate

# 4. Install dependencies
pip install -r requirements.txt

# 5. Set up environment variables (optional)
cp .env.example .env
# Edit .env file with your API keys if needed
```

### Launch Application

```bash
# Start the development server
python app.py

# Open your browser and navigate to:
# http://localhost:5000
```

## 🎯 Use Cases & Examples

### Example 1: Symptom Analysis
```
User Input: "I have chest pain and shortness of breath in Dallas"

FinHealth Response:
✅ Analyzes symptoms using AI
✅ Suggests relevant procedures (ECG, Chest X-ray, Stress Test)
✅ Finds hospitals in Dallas area
✅ Compares prices with insurance coverage
✅ Provides estimated wait times and ratings
```

### Example 2: Procedure Comparison
```
User Input: "I need an MRI scan in Chicago, I have Aetna insurance"

FinHealth Response:
✅ Finds 8 hospitals in Chicago area
✅ Compares MRI prices ($800-$2,400)
✅ Calculates Aetna coverage (estimated out-of-pocket: $480)
✅ Shows cash discount options (save $200-$400)
✅ Provides hospital ratings and availability
```
